Website for a cake recipe

I created this simple and light web application for ease for order items for my client.

The Objectives: Gain visibility for food lovers, display the food to users that want something similar and be able to place an order immediately.

This project is simple, light and easy to flow but educative and efficient for those wanting a food website for ease of ordering food. My client was very happy with the outcome of the project.

Technologies: HTML, CSS, JavaScript, Django, Python, WhoGoHost
