Website for newspaper

I created this project for a fellow student who is studying mass communication.

The Objectives: To be enable to register, login and logout users, create news, post news, update or edit news and delete the news by users.

My client was happy with the project as I designed an administration interface for him so as he can easily remove a user or delete inappropriate news created.

Technologies: HTML, CSS, BOOTSTRAP4, DJANGO, PYTHON, HEROKU
