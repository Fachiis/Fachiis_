Website for bookstore

I created this project for a bookstore owner.

The Objectives: To be enable to register, login, logout and carryout password change and reset, create new books for sales, update or edit book information and delete book by uploaders.

My client was happy with the project as I designed an administration interface for him so as he can easily add/remove a user or add/delete books among another functionalities

Technologies: HTML, CSS, BOOTSTRAP4, DJANGO, DOCKER, PYTHON, HEROKU
