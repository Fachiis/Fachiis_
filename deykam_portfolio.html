A social network application

This is one of the most challenging web application I was actively engaged in with some other developers. We created this application to help people connect with each other and share their thoughts and experiences. I created this project with a UI designer for a company located in another country. All communication was done via virtual meetings and we used the SCRUM methodology for development

The Objectives: a social network application where people including students can connect, do homework, project and have fun. We completed the first version of the application and the company was impressed with the project outcome.

Technologies: HTML, CSS, Bootstrap, Python, JavaScript, Django, Django REST, Django-Allauth, PostgreSQL, Pythonanywhere, Digital Ocean, Git, GitHub
